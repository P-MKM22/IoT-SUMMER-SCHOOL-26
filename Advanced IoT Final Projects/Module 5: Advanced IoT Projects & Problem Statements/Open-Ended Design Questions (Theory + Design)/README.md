# Open-ended Design Questions (Theory + Design)

---

# Overview

The **Open-ended Design Questions** are part of **Module 5 – Advanced IoT Final Project** of the **IoT Summer School 2026**. This section is designed to evaluate the ability to analyze real-world IoT problems, develop practical solutions, and demonstrate engineering design skills through both conceptual understanding and system-level design.

The contents are organized into two complementary sections. The **Theory Questions** focus on security analysis and learning reflection, allowing students to demonstrate their understanding of IoT concepts, challenges, and best practices. The **Advanced IoT Final Project** contains the design challenge, where complete IoT solutions are proposed for real-world Smart City applications using professional documentation, modular project architecture, and deployment-oriented system design.

Together, these directories provide a comprehensive assessment of both the theoretical knowledge and practical design capabilities developed throughout the IoT Summer School.

# Directory Contents

## Theory Questions

This directory contains **two theory-based questions** that assess conceptual understanding, security awareness, analytical thinking, and the overall learning experience gained during the IoT Summer School.

- **Q49 – Security Vulnerability Analysis**
- **Q50 – Reflection and Learning Portfolio**

---

## Advanced IoT Final Project

This directory contains the **design section** of the Open-ended Design Questions through **Q48 – Design Challenge: IoT for Jammu Smart City**.

The project includes **six independent Smart City IoT solutions**, each addressing a different real-world urban challenge. Every project is organized into two main components:

- **Code** – Complete modular source code developed for the proposed IoT solution.
- **Proposal.md** – Detailed project proposal including the project overview, problem statement, proposed solution, hardware requirements, software requirements, system architecture, data flow, working principle, expected outcomes, future scope, and author information.

The six Smart City project proposals included are:

- Intelligent Waste Management System
- Smart Clinic System
- Smart Fire Detection and Emergency Response System
- Smart Grocery Distribution System
- Smart Parking Availability and Management System
- Smart Street Lighting System
