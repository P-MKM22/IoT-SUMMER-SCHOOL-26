# Module 05: Advanced IoT Projects & Problem Statements

## Overview

Module 5 is the final implementation module under the **Advanced IoT Final Projects** directory of the **IoT Summer School 2026** repository. This module focuses on solving real-world IoT challenges through practical system design, problem-solving, and technical analysis. It covers multiple application domains including Smart Home, Security, Agriculture, Healthcare, and Smart City solutions.

Unlike the previous modules, Module 5 is divided into two separate parts:

- **Advanced IoT Final Projects** – Contains project-based implementation and design challenges (Q41–Q48).
- **Theory Questions** – Contains analytical and reflection-based questions (Q49–Q50) maintained in a separate directory.

## Question Overview

### Problem Statement Type A: Smart Home & Environment

| Question | Brief Description |
|----------|-------------------|
| **Q41. Smart Home Air Quality Monitor** | Designed an IoT system to monitor indoor air quality using environmental sensors and generate alerts when air quality degrades. |
| **Q42. Automated Plant Watering System** | Developed an automated irrigation solution that monitors soil moisture and controls watering without manual intervention. |

---

### Problem Statement Type B: Security & Access Control

| Question | Brief Description |
|----------|-------------------|
| **Q43. Smart Door Lock with OTP** | Designed a secure PIN and OTP-based smart door access system using keypad, Bluetooth communication, and automatic verification. |
| **Q44. PIR-Based Security Camera Trigger** | Implemented an intelligent surveillance concept that activates a camera automatically when motion is detected using a PIR sensor. |

---

### Problem Statement Type C: Agriculture & Environment
