## Commit Message Convention

| Prefix | Purpose |
|---------|---------|
| `feat:` | Introduces a new feature or implementation |
| `docs:` | Adds or updates documentation |
| `fix:` | Resolves bugs or implementation issues |

### Example Commit Messages

```text
docs: initialize repository structure
docs: add Week 01 documentation
feat: implement LED blinking project
fix: correct OLED display initialization
```
