# Overview

This section contains practical Arduino programming assignments completed as part of **Module 2: IoT Fundamentals & Arduino Programming** during the **IoT and Drone Building Summer School 2026**. The projects focus on embedded programming concepts such as digital I/O, PWM, Serial communication, finite state machines, and user interaction through sensors and actuators.

Each project has been developed, tested, and documented using **Arduino IDE** and **Wokwi**, with dedicated source code, documentation, hardware screenshots, demonstration videos, and simulation links.

---

# Projects Included

| Question | Project | Description |
|----------|---------|-------------|
| Q14 | Traffic Light Controller | Implemented a traffic light controller with pedestrian crossing functionality using LEDs and push buttons. |
| Q15 | Digital Piano using Buzzer | Designed a digital piano capable of playing major and minor musical scales using push buttons and a buzzer. |
| Q16 | Serial Command Interface | Developed a command-based Serial interface to control the built-in LED, execute blink operations, display system status, and validate user commands. |
| Q17 | PWM Fading Night Light | Created a PWM-based night light featuring breathing, fast pulse, and SOS lighting modes with push-button mode selection. |
| Q18 | State Machine Implementation | Implemented a vending machine finite state machine with multiple operating states, LED state indicators, button-based transitions, and Serial Monitor logging. |
