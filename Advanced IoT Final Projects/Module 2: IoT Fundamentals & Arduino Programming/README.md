# Overview

This directory contains the **practical implementation** of **Module 2: IoT Fundamentals & Arduino Programming**. It focuses on hands-on Arduino programming through real-world embedded system projects that strengthen the concepts introduced during the module.

The practical work for Module 2 is organized under **Section B: Programming Tasks**, where each project demonstrates a specific embedded programming concept such as digital I/O, PWM, Serial communication, state machines, and hardware interfacing.

> **Note:** The theoretical content of Module 2, including **Section A (Q11–Q13)** and **Section B (Q19–Q20)**, is maintained separately in the **Theory Questions** directory.
