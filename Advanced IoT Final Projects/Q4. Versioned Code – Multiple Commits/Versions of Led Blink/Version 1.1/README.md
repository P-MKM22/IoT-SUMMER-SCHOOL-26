# Version 1.1 – LED Blinking with Serial Monitor

## Project Overview

Version 1.1 extends the basic LED blinking project by displaying status information on the Serial Monitor while maintaining the original blinking functionality.

---

## Objective

- Learn Serial Monitor communication.
- Monitor program execution.
- Improve debugging skills.

---

## Components Used

- Arduino Uno
- LED
- 220 Ω Resistor
- Breadboard
- Jumper Wires
