# Version 1.2 – Potentiometer Controlled LED Blinking

## Project Overview

Version 1.2 introduces analog input by using a potentiometer to control the LED blinking speed. The delay changes dynamically according to the potentiometer position.

---

## Objective

- Learn analog input.
- Control blink speed dynamically.
- Read sensor values using the ADC.

---

## Components Used

- Arduino Uno
- LED
- 220 Ω Resistor
- Potentiometer
- Breadboard
- Jumper Wires
