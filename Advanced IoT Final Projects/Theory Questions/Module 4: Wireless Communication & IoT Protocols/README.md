# Overview

Module 4 introduces the communication technologies and networking protocols that form the backbone of modern Internet of Things (IoT) systems. It focuses on secure, scalable, and energy-efficient communication between IoT devices, gateways, cloud platforms, and end-user applications.

The theory section of this module covers **Protocol Theory**, providing an in-depth understanding of widely used IoT communication protocols, wireless security mechanisms, Low-Power Wide Area Networks (LPWANs), IoT gateways, and MQTT Quality of Service (QoS). These concepts are fundamental for designing reliable, secure, and large-scale IoT deployments.

# Key Takeaways

- MQTT is optimized for lightweight, low-power IoT communication, whereas HTTP is better suited for web-based services and REST APIs.
- Strong Wi-Fi security practices, including WPA3, TLS encryption, and secure credential management, are essential for protecting IoT devices.
- LPWAN technologies such as LoRa enable long-range communication while maintaining low power consumption, making them ideal for smart agriculture and smart city applications.
- IoT gateways act as intermediaries between sensor networks and cloud platforms by performing protocol translation, edge processing, and secure data forwarding.
- MQTT QoS levels allow developers to balance reliability, bandwidth, latency, and power consumption according to application requirements.
