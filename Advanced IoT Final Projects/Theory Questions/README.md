# Theory

## Overview

This folder contains all the theoretical questions and their respective answers completed as part of the **IoT and Drone Building Summer School 2026** conducted by the **Indian Institute of Technology Jammu (IIT Jammu)**.

The theory section covers fundamental concepts related to Git, GitHub, Internet of Things (IoT), software development, version control, networking, embedded systems, and other topics included in the course curriculum.

Each question is documented in a clear, structured, and concise manner to provide a comprehensive understanding of the concepts while serving as a reference for future learning.
