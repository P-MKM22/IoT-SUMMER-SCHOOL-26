# Overview

This section contains the theoretical questions associated with **Section B – Programming Tasks** of **Module 2: IoT Fundamentals & Arduino Programming**. These questions explain the fundamental Arduino programming concepts that support the practical implementation of embedded and IoT applications.

The topics covered include analog and digital operations, Pulse Width Modulation (PWM), Arduino program execution flow, blocking and non-blocking programming, and their practical significance in IoT systems.
