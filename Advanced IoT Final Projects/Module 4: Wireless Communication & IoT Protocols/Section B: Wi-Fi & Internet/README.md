# Section A – Bluetooth Projects

## Overview

Section A focuses on the implementation of **Bluetooth-based wireless communication** for Internet of Things (IoT) applications. The projects in this section introduce both **Bluetooth Classic (HC-05)** and **Bluetooth Low Energy (BLE)**, enabling embedded systems to communicate wirelessly with external devices.

The activities are designed to build practical knowledge of wireless device control, environmental data transmission, and BLE architecture. These projects serve as the foundation for understanding short-range communication technologies commonly used in modern IoT systems.

---

## Projects Included

| Project | Description |
|---------|-------------|
| **Bluetooth LED Controller (HC-05 + Arduino)** | Demonstrates wireless LED control using Bluetooth commands received from a mobile device. In simulation, the Serial Monitor is used as an alternative to the HC-05 module while preserving the same command structure. |
| **Bluetooth Sensor Dashboard (ESP32 BLE)** | Demonstrates a BLE-based environmental monitoring system that periodically acquires temperature and humidity data and prepares it for wireless transmission using BLE Services and Characteristics. |

---

## About This Section

Through these projects, learners gain hands-on experience with:

- Bluetooth Classic communication
- Bluetooth Low Energy (BLE)
- ESP32 wireless capabilities
- Remote device control
- Wireless sensor data transmission
- BLE GATT architecture
- Practical IoT communication workflows

These experiments establish the foundation required before progressing to **Wi-Fi communication, MQTT messaging, and REST API integration** in the following sections.
