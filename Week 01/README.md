# Week 01 – Foundations of IoT, AI, Cybersecurity, ESP32, and GitHub

## Overview

The first week of the **IoT Summer School 2026** focused on building a strong foundation in the Internet of Things (IoT), Artificial Intelligence (AI), cybersecurity, embedded systems, and version control using GitHub. The sessions introduced the architecture of IoT systems, communication protocols, embedded hardware, ESP32 programming, and the role of cybersecurity and eBPF in securing IoT devices. Practical activities included Arduino programming, ESP32 configuration, LED blinking, and GitHub repository management.
