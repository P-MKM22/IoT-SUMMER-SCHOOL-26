<h1 align="center">Week 01 - Introduction to Arduino and LED Blinking</h1>

<h2>Overview</h2>
<p>
This week focused on learning the fundamentals of Arduino programming and embedded systems.
The primary experiment involved controlling an LED using an Arduino Uno to understand digital output operations and the basic structure of an Arduino program.
</p>

<h2>Objectives</h2>
<ul>
  <li>Understand the Arduino IDE.</li>
  <li>Learn the <code>setup()</code> and <code>loop()</code> functions.</li>
  <li>Configure digital output pins.</li>
  <li>Control an LED using <code>digitalWrite()</code> and <code>delay()</code>.</li>
</ul>

<h2>Experiment</h2>
<h3>LED Blinking</h3>
<p>
The Arduino program turns the LED ON for one second and OFF for one second continuously, demonstrating basic digital output control.
</p>

<h2>Components Used</h2>
<ul>
  <li>Arduino Uno</li>
  <li>LED</li>
  <li>220Ω Resistor</li>
  <li>Breadboard</li>
  <li>Jumper Wires</li>
  <li>USB Cable</li>
</ul>

<h2>Learning Outcomes</h2>
<ul>
  <li>Learned the structure of an Arduino program.</li>
  <li>Understood digital output control.</li>
  <li>Successfully uploaded and tested an Arduino program.</li>
  <li>Built a foundation for future IoT projects.</li>
</ul>
