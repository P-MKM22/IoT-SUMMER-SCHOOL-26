# Week 03 – REST API Integration, Live Weather Monitoring, Sensor Interfacing, Actuator Control, Drone Technology, Final Project Development, and Project Presentation

## Overview

Week 03 focused on applying IoT concepts through practical implementation of REST API communication, environmental monitoring, sensor interfacing, actuator control, embedded programming, drone technology, and final project development. During this week, we integrated ESP32 with online weather services, developed multiple sensor- and actuator-based Arduino projects, explored drone architecture and flight control systems, and completed the final IoT project.

---

## Learning Objectives

- Understand REST APIs and JSON data handling.
- Integrate ESP32 with live weather services.
- Interface environmental sensors with Arduino and ESP32.
- Build real-time monitoring systems.
- Learn actuator control using servo motors.
- Implement structured sensor data logging.
- Apply non-blocking programming using `millis()`.
- Understand drone architecture and flight control systems.
- Study GPS navigation and Mission Planner software.
- Develop and present a complete IoT project.
