# Classwork

## Live Weather Monitoring System

As a classroom activity, we implemented a **Live Weather Monitoring System** using the **ESP32**, **DHT11 Temperature and Humidity Sensor**, and the **Open-Meteo Weather API**.

The objective was to understand how IoT devices can communicate with cloud-based web services to retrieve real-time weather information and compare it with locally measured environmental data.

### Components Used

* ESP32 Development Board
* DHT11 Temperature and Humidity Sensor
* OLED Display
* Wi-Fi Network
* Open-Meteo Weather API

### Concepts Applied

* Wi-Fi connectivity using ESP32
* HTTP GET Requests
* REST API Integration
* JSON Parsing
* Displaying weather information on OLED
* Comparing local sensor readings with online weather data
* Debugging API responses

# Final Project Development

## Project Title

**IoT-Based Mortal Kombat Game using ESP32, OLED Display, LEDs, and MPU6050 IMU**

### Project Description

The final project involved developing a gesture-controlled fighting game using the ESP32 microcontroller. Player gestures were detected using the MPU6050 IMU sensor and processed by the ESP32 to control character movements, attacks, and animations displayed on the OLED screen. LEDs were incorporated to provide visual feedback during gameplay.

### Components Used

* ESP32 Development Board
* OLED Display
* MPU6050 IMU Sensor
* LEDs
* Breadboard
* Jumper Wires
* USB Cable
