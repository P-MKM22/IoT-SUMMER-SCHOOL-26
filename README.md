# Final Project

## IoT-Based Gesture Controlled Mortal Kombat Game using ESP32, OLED Display, LEDs and MPU6050 IMU

### Project Description

This project demonstrates a gesture-controlled fighting game developed using the ESP32 microcontroller and MPU6050 IMU sensor. Player gestures are detected in real time to perform attacks, movements, and animations on an OLED display. The project integrates embedded programming, sensor interfacing, animation, and IoT concepts learned throughout the program.
